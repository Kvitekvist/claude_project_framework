## Purpose

This framework provides a standardized development workflow for all AI-assisted software projects.

The goals are:

* Consistent architecture
* Repeatable workflows
* Long-term maintainability
* Reliable documentation
* Traceable development history

---

## Framework Components

* Claude operating instructions
* Ticket system
* Project memory
* Architecture tracking
* Build automation
* Git workflow
* Documentation standards

---

## Migration Rules

When a newer framework version is detected:

1. Compare framework versions.
2. Suggest improvements.
3. Never overwrite project-specific files without approval.
4. Preserve backwards compatibility whenever possible.
